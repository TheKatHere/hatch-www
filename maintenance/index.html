What is Hatch.lol?

Simply put, Hatch.lol is a third-party project hosting service for the Scratch programming language.

Scratch is a great option to learn the fundamentals of programming, but it is designed for young children. Many Scratchers understandably find themselves outgrowing Scratch as they get older, and/or have distaste for new Scratch features, but they want to keep using the Scratch language. Hatch.lol is designed with this group of users in mind. While we're very similar to Scratch, having a close online community, we have more relaxed rules.
